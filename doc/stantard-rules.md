## Standard (Seven-segment) Matchstick Rules

| Character | Matchsticks | Move (one match) | Add (one match) | Remove (one match) |
|-----------|------------:|---------------------|--------------------|------------------------|
| (space)   | 0           |                     | -                  |                        |
| -         | 1           |                     | +, =               | (space)                |
| +         | 2           | 1, =                |                    | -                      |
| x         | 2           | /                   |                    |                        |
| /         | 2           | x                   |                    |                        |
| =         | 2           | +                   |                    | -                      |
| 1         | 2           | +                   | 7                  |                        |
| 7         | 3           |                     |                    | 1                      |
| 4         | 4           | 11                  |                    |                        |
| 11        | 4           | 4                   |                    |                        |
| 2         | 5           | 3                   |                    |                        |
| 3         | 5           | 5, 2                | 9                  |                        |
| 5         | 5           | 3                   | 9, 6               |                        |
| 0         | 6           | 6, 9                | 8                  |                        |
| 6         | 6           | 0, 9                | 8                  | 5                      |
| 9         | 6           | 0, 6                | 8                  | 3, 5                  |
| 8         | 7           |                     |                    | 0, 6, 9                |
